What is NativeScript


  • Open Source, backed by Progress
  • Multiple flavors (Vanilla, Angular, and Vue)
  • Builds truly native apps?
    • Access to all native objc/Java classes
    • Provides native performance
    • Enables the same customization as native

How it Works

You write JS/TS code

It runs inside of the device's JavaScript VM

  • Android - V8
  • iOS - JavaScriptCore

NS generates metadata which tells what native API's are available

The JS code is ran through a type conversion process to convert it to native

The converted code is sent to the call dispatcher to call the native API's

Why NativeScript?


  • 100% sharable code across platforms (Android and iOS)
  • Can use existing web development knowledge
  • Allows you to use existing libraries
